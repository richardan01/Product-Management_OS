# Stakeholder memory — compact pointers (approval-gated)

Compact, durable pointers to the people the assistant should understand. The
**canonical** stakeholder profiles live in `Knowledge/People/` — this file is a
lightweight index plus a few durable working-style notes.

> **Approval gate.** Sensitive stakeholder / company / customer facts require
> **explicit user approval** before they are written here. This is a public
> template — ship placeholders only. Keep real names and private notes in a
> private fork or local copy. See `MEMORY_POLICY.md`.

## Format

```
### [STAKEHOLDER_NAME] — [role]

- **Working-style note:** [durable, non-sensitive]
- **Canonical profile:** `Knowledge/People/[name].md`
```
