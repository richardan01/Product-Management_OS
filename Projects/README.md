# Projects/

One folder per project or workstream. Every project starts with a `brief.md` — the OS convention (`CLAUDE.md` DO-NOT list) is that no other project file is created before the brief exists.

```
Projects/
└── <your-project>/
    ├── brief.md          # start here — use Templates/project-brief.md
    ├── prd.md            # Templates/PRD.md or Templates/PRD-AI-feature.md
    ├── discovery/        # research inputs
    └── synthesis/        # research outputs (/synthesize-research)
```

Project content is yours and typically private — keep real project folders in your private fork or local copy. This README ships with the template so the path the docs reference exists on a fresh clone.
