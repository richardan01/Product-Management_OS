## How to Set Up an MCP

1. Find the MCP server for your tool (check [MCP registry](https://modelcontextprotocol.io))
2. Add it to your Claude Code config: `~/.claude/claude_desktop_config.json`
3. Document it here with key commands

## Useful MCPs for a [Your Role]

- **Google Calendar** — let Claude check your schedule during standup
- **Slack** — read threads, draft messages
- **Google Sheets / Notion** — pull in roadmap or metrics data
- **Linear / Jira** — connect tasks to engineering tickets
