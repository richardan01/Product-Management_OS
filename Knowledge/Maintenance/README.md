# Knowledge/Maintenance — weekly sweep logs

`/wiki-maintain` writes its weekly sweep report here as `YYYY-MM-DD-sweep.md` (run every Friday).

Each report covers the 9 checks: stale knowledge, evidence decay, hypothesis hygiene, stakeholder cadence, strategy tension, knowledge synthesis, archival sweep, compression health, and decision outcome / reversal-condition audit. Items are flagged 🔴 action needed / 🟡 watch / ✅ clean.
