# Gate Group — Results (local only)

Eval **run logs are not committed to this public template.** They contain run-specific
output (model responses, transcripts, scores) that belongs in your private working copy,
not in the shared template.

Write run logs here as `YYYY-MM-DD_<label>.md` following the format in
`../README.md` and the shared protocol in `../../README.md`. Everything in this
directory except this file is gitignored (see repo-root `.gitignore`).
