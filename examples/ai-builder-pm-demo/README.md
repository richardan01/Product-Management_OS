# AI Builder PM Demo (Sanitized)

## Example goal
Ship an AI-assisted onboarding feature that improves activation from signup to first value within 14 days.

## Example active task
Draft and align on an AI feature PRD covering user flow, model behavior, failure modes, and rollout guardrails.

## Example AI feature PRD use case
Use `Templates/prd-ai-feature.md` to define:
- user problem and target segment
- AI behavior and fallback behavior
- eval plan and launch gates
- risks, mitigations, and post-launch monitoring

## Example eval/review loop
1. Draft PRD from template.
2. Run `/build-review` for PM + build quality.
3. Run `/eval-review` for evaluation rigor and model-risk coverage.
4. Run `/go-nogo` before launch recommendation.
